贪吃蛇实现中的重要方法：

1.边界检测方法    isOnBorder();

2.蛇身的移动方法    snakeMove();
	在定时器中的执行方法中，先计算蛇头的下一个位置，然后把蛇身中前一个div的位置赋给后一个，
	重新绘制整个snake

3.生成食物的方法    createFood();
	随机生成食物的位置，并检测位置的有效性，若位置无效，则重复生成，若位置有效，则将食物加入dom

4.吃食物的方法    eatFood(food);
	将食物的div从dom中移除，并调用生成食物的方法重新生成食物

5.碰撞检测方法    collisionCheck();
